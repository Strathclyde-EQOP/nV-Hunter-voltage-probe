# nV-Hunter-voltage-probe

This repository contains design files for a high gain (80 dB, 10k), low bandwidth (0.1 - 200 Hz) voltage probe.

# Specification brief

- BW : 0.1 - 200 Hz
- RMS noise 38.1 nV 0.1 Hz - 5 MHz
- Peak-to-Peak noise 310.6 nV 0.1 Hz - 5 MHz
